# Security policy

If you discover a security vulnerability affecting **rock-capital-partners.com** or this repository, please report it privately to **info@rock-capital-partners.com**.

We aim to acknowledge reports within 5 business days. Please do not disclose the issue publicly before we have remediated.

In scope:
- The live site at https://rock-capital-partners.com
- This repository (zzzz-mike/rock-capital-partners-website)

Out of scope:
- Social-engineering, phishing, or physical attacks against the owners
- Third-party services we depend on (GitHub Pages, Mailgun, our domain registrar) — please report directly to those providers
